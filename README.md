# 🧹 Smart File Organizer

A Python-based automation tool designed to clean up messy directories instantly. This script intelligently categorizes files based on their extensions and moves them into dedicated folders.

## 🚀 Features
- **Dynamic Path Selection:** Users can input any directory path to organize.
- **Auto-Folder Creation:** Automatically creates "Images", "Documents", and "Programs" folders if they don't exist.
- **Dictionary Mapping:** Easy to scale and add more file extensions.
- **Safe Operations:** Uses `shutil` and `os` for reliable file transfers.

## 🛠️ Supported Formats
- **Documents:** `.pdf`, `.docx`
- **Images:** `.png`, `.jpg`
- **Programs:** `.exe`

## ⚙️ How to Use
1. Run the script.
2. Enter the full path of the folder you wish to organize.
3. Watch the magic happen!
