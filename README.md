# 🧹 Smart File Organizer (Dynamic Edition)
*Scroll down for Arabic | النسخة العربية بالأسفل* 👇

A powerful, dynamic Python automation tool that instantly cleans up messy directories. Instead of relying on predefined categories, this script intelligently reads the extension of ANY file and automatically creates a dedicated folder for it.

## 🚀 Features
- **Fully Dynamic Sorting:** Creates folders on-the-fly based on file extensions (e.g., `.pdf` goes to `PDF/`, `.mp4` goes to `MP4/`).
- **No Hardcoded Rules:** Supports literally any file format without needing code updates.
- **Safe Operations:** Ignores sub-directories to prevent recursive loops and safely moves files using `shutil`.
- **Standalone App:** Comes with an executable `.exe` file for instant use without needing Python installed.

## ⚙️ How to Use
1. Download the `main.exe` from the `dist` folder, or run the `main.py` script.
2. Enter the full path of the cluttered folder when prompted.
3. Watch the magic happen! Every file type will get its own organized folder.

# 🧹 منظم الملفات الذكي (النسخة الديناميكية)

أداة أتمتة قوية وديناميكية مبرمجة بلغة بايثون لتنظيف وترتيب المجلدات المبعثرة في ثوانٍ. بدلاً من الاعتماد على تصنيفات مبرمجة مسبقاً، يقوم هذا السكربت بقراءة صيغة أي ملف بذكاء وينشئ مجلداً خاصاً له تلقائياً.

## 🚀 المميزات
- **ترتيب ديناميكي بالكامل:** ينشئ المجلدات فورياً بناءً على صيغة الملف (مثال: ملفات `.pdf` تذهب لمجلد `PDF/`، وهكذا).
- **لا توجد قواعد ثابتة:** يدعم حرفياً أي صيغة ملف في العالم بدون الحاجة لتحديث الكود.
- **عمليات آمنة:** يتجاهل المجلدات الفرعية لمنع الأخطاء المتداخلة، وينقل الملفات بأمان تام باستخدام مكتبة `shutil`.
- **تطبيق مستقل:** يتوفر كملف تنفيذي `.exe` جاهز للاستخدام المباشر بدون الحاجة لتثبيت بايثون على جهازك.

## ⚙️ طريقة الاستخدام
1. قم بتحميل ملف `main.exe` من مجلد `dist`، أو قم بتشغيل سكربت `main.py`.
2. أدخل المسار الكامل للمجلد الذي تريد تنظيفه عندما يُطلب منك ذلك.
3. استمتع بالسحر! كل صيغة ملف ستحصل على مجلدها المستقل والمرتب.
